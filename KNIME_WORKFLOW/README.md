# KNIME Workflows

This folder hosts KNIME workflows for dataset processing and multi-stage filtering.

## Workflows Included

### 1. RET_DATASET.knwf
Dataset collection workflow to process and curate the RET mutant datasets.

### 2. SINGLE_OBJECTIVE_MULTI_STAGE_FILTER.knwf
Multi-stage filtering workflow for single-objective REINVENT optimization results.

### 3. MULTI_OBJECTIVE_MULTI_STAGE_FILTER.knwf
Multi-stage filtering workflow for multi-objective REINVENT optimization results.
